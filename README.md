# The Rooted Parcel 🌿

A beautifully designed, sustainable and eco-friendly e-commerce web application.

## Features
- **Sustainable Products**: Provides curated collections for Home & Living, Personal Care, Reusable Kitchen, and Fashion.
- **Dynamic Cart**: Full cart management with real-time total calculation.
- **Responsive Design**: Modern, responsive layout that works seamlessly across all devices.
- **User Dashboard**: Track orders, handle returns, and manage user details.
- **Admin Dashboard**: A secure environment to oversee products, manage orders, and process user returns.
- **Optimized for SEO**: Includes meta tags, Open Graph, Twitter cards, structured data, a dynamically mapped `sitemap.xml`, and explicit searchbot directives in `robots.txt`.

## Tech Stack
- HTML5, Vanilla JavaScript, Vanilla CSS
- Hosted on Firebase

## Local Development
1. Clone the repository: `git clone https://github.com/Atri02/ECO-WEB.git`
2. Open the directory: `cd ECO-WEB`
3. Serve locally (e.g., using Live Server extension in VS Code)
